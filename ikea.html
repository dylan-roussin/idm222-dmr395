IKEA Case Study

For my User Interface Design I class, the goal was to choose a website that could be improved with a redesign. I choose IKEA’s website due to a couple factors like cluttered navigation and item pages, as well as some empty feeling pages.

Navigation

One of the main issues I found with was the cluttered navigation. Dropdown menu’s were so full of content that you needed to scroll in order to see all of it. My goal was to declutter the navigation and consolidate categories to make it far easier to browse and use the website.

I used card sorting in order to make the website easier to navigate, find which paths would be best to consolidate together, and how the navigation should be structured. I interviewed three people and had them complete the activity.
